# BigIntLib - Библиотека для работы с большими числами

## 📝 Описание
Курсовая работа по реализации библиотеки для работы с целыми и вещественными числами размером 256-512 бит. 
Библиотека позволяет выполнять арифметические операции с числами.

## 🚀 Возможности
- Базовые арифметические операции (сложение, вычитание, умножение, деление)
- Побитовые операции
- Сравнение чисел
- Конвертация из/в строку
- Поддержка различных систем счисления

## 🛠️ Технические требования

- .NET Framework 4.7.2 или выше
- C# 7.0+ (поддерживается в .NET Framework 4.7.2)
- Visual Studio 2017 версии 15.7 или новее / VS 2019/2022

## 🔧 Сборка и запуск

### Через Visual Studio:
1. Откройте нужный `.sln` файл:
   - `lib/KURSOVAYA.sln` - только библиотека
   - `app/KURSOVAYA_APP.sln` - демо-приложение
   - `tests/KURSOVAYA_TESTS.sln` - тесты
2. Выберите конфигурацию `Release` или `Debug`
3. Нажмите `Build` → `Build Solution` (Ctrl+Shift+B)

### Через командную строку:
```bash
# Сборка библиотеки
cd lib
dotnet build KURSOVAYA.sln

# Сборка приложения
cd ../app
dotnet build KURSOVAYA_APP.sln

# Запуск тестов
cd ../tests
dotnet test KURSOVAYA_TESTS.sln
